# Интерактивная карта Нагорного парка во Владивостоке
---

### Концепция проекта

Проект представляет собой интерактивную карту парка, созданную с использованием API Яндекс.Карт.
Цель проекта — визуализировать территорию парка с помощью современных веб-технологий и научиться работать с картографическими API, пользовательскими интерфейсами и структурой данных.

### Карта должна:

- Показывать основные зоны парка (поляны, игровые зоны, озеро, фудкорт и т.п.);
- Иметь метки интересных мест (детская площадка, туалет, кафе и т.д.);
- Позволять пользователю кликать по элементам карты и получать подробную информацию о выбранной точке;
- Иметь меню (сбоку или сверху), из которого можно выбрать объект и перейти к нему на карте.

### Технические требования

#### 1. Основной стек технологий

- HTML5 / CSS3 / JavaScript
- Bootstrap 5 (для адаптивного интерфейса и offcanvas-меню)
- Yandex.Maps API 2.1

#### 2. Функциональные требования

**Карта**:

- Отображается через Яндекс.Карты API.
- Центрирована на нужных координатах парка.
- Установлены ограничения перемещения, чтобы пользователь не «улетел» за пределы парка.
- Добавлены контролы карты (масштаб, тип карты, полноэкранный режим, геолокация).

**Зоны и объекты**:

- Несколько полигонов (зоны парка) с разным цветом заливки.
- Маршруты / дорожки обозначены линиями (Polyline).
- Несколько меток (Placemark) с иконками, названиями и описанием.
- Всплывающие окна (balloons)
- При клике на метку открывается всплывающее окно с фотографией и описанием объекта.
- Данные о метках хранятся в массиве places (JS-объекты с полями coords, name, img, desc, iconUrl).

**Боковое меню**:

- Реализовано через Bootstrap Offcanvas.
- Содержит список всех точек интереса.
- При выборе элемента — карта перемещается к объекту, и открывается его balloon.
- В меню предусмотрен режим просмотра информации с кнопкой “Назад”.

**Адаптивность**:

- Интерфейс корректно отображается на экранах от смартфона до ноутбука, но имеет требование к mobile first.
- Offcanvas открывается на весь экран на мобильных устройствах.
